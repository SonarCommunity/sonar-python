Variables, Classes and functions should be defined before they are used, otherwise the code will fail.

Noncompliant Code Example

unknown_var  # Noncompliant (variable is never defined)

foo()  # Noncompliant
foo = sum

func()  # Noncompliant
def func():
    pass

MyClass()  # Noncompliant
class MyClass:
    pass

myglobal = 42
def use_global():
    print(myglobal)  # Noncompliant
    global myglobal

Compliant Solution

foo = sum
foo()

def func():
    pass
func()

class MyClass:
    pass
MyClass()

myglobal = 42
def use_global():
    global myglobal
    print(myglobal)
